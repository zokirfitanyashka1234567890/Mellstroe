Cascading Style Sheets Css

Css veb-sahifa yaratuvchilari tomonidan ranglar,shriftlar,alonida bloklarning tartibini va ushbu veb-sajifalarning paydo bolishning boshqa jihatlarnin beligilash uchun ishlatiladi.Css-ni ishlab chiqishning asosiy maqsadi veb-sahifaning[HTML yoki boshqa belgilash tillari yordamida ishlab chiqarilgan]muntiqiy tuzulishinig tavsifini ushbu veb sahifa[hozir rasmiy Css tilli yordamida ishlab chiqarilgan] tashqi korishini tavsinida ajratish edi.Ushbu ajratiz hijatning mavjudligini oshirishni,koproq maslashuvckiling va taqdimoling boshqarishqobilyatni taminlab,shuninbek tarkibiy tarkibdagi murikablik va takrovchiling kamaytirish mumkun.
